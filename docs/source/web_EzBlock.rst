网页版 EzBlock
===========================================


进入 `网页版 EzBlock <http://EzBlock.com.cn/EzBlock_studio/index.html?lang=zh-hans&distributor=mammoth>`_ 。

.. note:: 建议使用Chrome浏览器。

在首页点击左上角的产品按钮。

.. image:: image/sp211203_160245x.png

点击连接。

.. image:: image/sp211203_160318x.png

输入IP地址。

.. image:: image/sp211203_160509x.png

随后将弹出“连接成功”的提示，此时你可以回到 :ref:`EzBlock 项目` 。

.. image:: image/sp211203_160543x.png
